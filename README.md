# webpack4Hanzilu #

一个自用的webpack脚手架

功能：

- 实现生产配置和开发配置分离
- 按模板自动生成html
- 图片路径转换与压缩
- ES6语法与新增类函数支持
- js、css、less、html的打包压缩
- 生产环境时,css抽离成独立的css
- 本地开发服务器搭建及热模块替换
- 给文件加上hash值
- 生成文件前清空输出文件夹
- source-map定位源文件
- 集成eslint检测和editorconfig配置
- 集成react,react-dom
- 集成react-router-dom
- 集成React的UI组件库antd-mobile
- 集成字体图标库Font Awesome
- 单元测试采用jest,enzyme
